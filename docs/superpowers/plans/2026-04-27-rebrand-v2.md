# CaliLean Rebrand v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the "Salt & Iron" warm palette, Fraunces/Inter typography, and old logo with a white + carolina blue + black palette, Plus Jakarta Sans, new PNG wordmark, and new product imagery.

**Architecture:** Token-first approach — update CSS custom properties and Tailwind config so ~70% of components auto-cascade. Then swap assets (logo PNGs, favicon, product images) and fix the ~6 components with hardcoded colors. Finally update brand docs and email constants.

**Tech Stack:** Next.js 15, Tailwind CSS 3, next/font/google (Plus Jakarta Sans), React 19, Medusa email templates
